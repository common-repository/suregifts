== Description ==

The Suregifts plugin creates a field on the cart page of your Store/Website that  validates Suregifts vouchers and apply them as a discount to a purchase by customers with gift vouchers from Suregifts.


== Installation ==
1.	Upload 'SureGifts WooCommerce' to the '/wp-content/plugins/' directory
2.	Activate the plugin through the 'Plugins' menu in WordPress Plugins Area
3.	Fill in the merchant credentials shared with you to authenticate the plugin. 
4. 	If you are yet to be a registered merchant on Suregifts, send an email to 	NG - bizdev@suregifts.com.ng/admin@suregifts.com.ng 
	KE - bizdev@suregifts.co.ke/admin@suregifts.co.ke

== Frequently Asked Questions ==

Yes
